🎯 ЧТО ДЕЛАТЬ (5 минут):


ШАГ 1: Открой Render Dashboard
─────────────────────────────────────────────────────────────
   1. Зайди на: https://dashboard.render.com
   2. Войди в аккаунт (если не залогинен)


ШАГ 2: Найди PostgreSQL сервис
─────────────────────────────────────────────────────────────
   В списке сервисов найди:

   ⚪ monolit-db (PostgreSQL)

   Кликни на него!


ШАГ 3: Открой Shell (консоль базы данных)
─────────────────────────────────────────────────────────────
   Вверху страницы увидишь кнопки:

   [ Info ]  [ Shell ]  [ Metrics ]  [ Settings ]

   Кликни на → [ Shell ]

   Откроется чёрный экран (терминал)


ШАГ 4: Подключись к базе
─────────────────────────────────────────────────────────────
   В Shell введи команду:

   psql -U monolit_user -d monolit_planner

   Нажми Enter

   Увидишь приглашение:
   monolit_planner=#


ШАГ 5: Скопируй SQL скрипт
─────────────────────────────────────────────────────────────
   Открой файл:  БЫСТРОЕ_РЕШЕНИЕ.sql

   Скопируй ВСЁ содержимое (Ctrl+A → Ctrl+C)


ШАГ 6: Вставь в Shell и выполни
─────────────────────────────────────────────────────────────
   В чёрном экране Shell:

   1. Кликни правой кнопкой → Paste
      (или Ctrl+Shift+V)

   2. Нажми Enter

   3. Подожди 2-3 секунды


ШАГ 7: Проверь результат
─────────────────────────────────────────────────────────────
   Увидишь таблицу:

    id | ai_enabled |        updated_at
   ----+------------+---------------------------
     1 | true       | 2026-01-21 12:34:56.789

   Если ai_enabled = TRUE → ✅ ВСЁ РАБОТАЕТ!


ШАГ 8: Проверь в браузере
─────────────────────────────────────────────────────────────
   1. Открой: https://monolit-planner-frontend.onrender.com

   2. Нажми Ctrl+Shift+R (жёсткая перезагрузка)

   3. Открой проект → таблица позиций

   4. В колонке "Dny" → ЗЕЛЁНАЯ КНОПКА ✨

   🎉 ГОТОВО!

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ


Q: Где найти Dashboard?
A: https://dashboard.render.com → залогинься → выбери "monolit-db"


Q: Не вижу кнопку Shell?
A: Убедись что открыл именно PostgreSQL сервис (monolit-db),
   а не веб-сервис (monolit-planner-api)


Q: Shell не открывается?
A: Попробуй другой браузер (Chrome, Firefox)
   Или используй альтернативный способ (напиши мне!)


Q: После выполнения SQL ничего не вышло?
A: Скопируй всё что написано в Shell и пришли мне -
   я посмотрю в чём ошибка


Q: Кнопки всё ещё нет после перезагрузки?
A: Попробуй в режиме инкогнито (Ctrl+Shift+N)
   Или очисти кеш браузера (Ctrl+Shift+Delete)
