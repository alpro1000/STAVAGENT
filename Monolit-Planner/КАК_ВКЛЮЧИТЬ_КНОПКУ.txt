🎯 САМЫЙ ПРОСТОЙ СПОСОБ (1 клик):

   1. Открой файл:  ENABLE_AI_BUTTON.html
   2. Нажми на большую кнопку
   3. Всё! Готово!


📝 ЧТО ПРОИЗОЙДЁТ:

   ✅ Включится feature flag FF_AI_DAYS_SUGGEST
   ✅ В Monolit Planner появится зелёная кнопка ✨
   ✅ Можно кликать для AI-подсказок по времени работ


🔍 ГДЕ НАЙТИ КНОПКУ ПОСЛЕ ВКЛЮЧЕНИЯ:

   1. Открой: https://monolit-planner-frontend.onrender.com
   2. Нажми Ctrl+Shift+R (перезагрузить страницу)
   3. Открой любой проект → таблица позиций
   4. Колонка "Dny" → справа от поля ввода → ✨ (зелёная)


💡 ЧТО ДЕЛАЕТ КНОПКА ✨:

   • Нажми ✨ → AI проанализирует тип работы, объём, нормы
   • Покажет подсказку: "2.5 дней (KROS 2024, 92% уверенность)"
   • Объяснение: "Для бетонирования 100 m³ с партой 4 человека..."
   • Можешь принять или изменить вручную

🛠️ АЛЬТЕРНАТИВНЫЕ СПОСОБЫ (если HTML не работает):


СПОСОБ 2: Bash-скрипт
   ./enable-ai-button.sh


СПОСОБ 3: Консоль браузера
   1. Открой Monolit Planner
   2. F12 (консоль разработчика)
   3. Вставь код:

   fetch('/api/config', {
     method: 'POST',
     headers: {'Content-Type': 'application/json'},
     body: JSON.stringify({
       feature_flags: {FF_AI_DAYS_SUGGEST: true}
     })
   }).then(r => r.json()).then(console.log).then(() => location.reload());

   4. Enter → перезагрузится страница → готово!

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ:

Q: Кнопки всё ещё нет после включения?
A: Нажми Ctrl+Shift+R для жёсткой перезагрузки (сбросит кеш)

Q: HTML файл не открывается?
A: Используй способ 3 (консоль браузера) - работает везде

Q: Ошибка "API is sleeping"?
A: Это нормально! HTML файл автоматически разбудит API и повторит попытку

Q: Сколько времени займёт?
A: 10-60 секунд (зависит от Render Free Tier cold start)
