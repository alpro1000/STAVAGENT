# Zadání majitele — rozsah prací (scope)

> Přepis zadání (RU/CZ), zmraženo jako vstup. Mapováno na sekce S1–S10 v atomic worklistu.

- **Elektrika** — kompletní výměna, + orientační cena s materiálem + revize. → **S6**
- **Potolky (stropy)** — sádrokarton všude **kromě druhého světa** (dvojvýška), cca −35 m² z celkových stropů. → **S5**
- **Stěny** — sítka + lepidlo + štuk. → **S1** (+ malba = povinný finiš, v nabídce mistra CHYBÍ)
- **Paркet** — reštaurace (broušení + lak). → **S4**
- **Nové podlahy** — vinyl v ostatních místnostech. → **S3**
- **Sanuzly** — kompletní demontáž + montáž, plitka jiná, sprcha předělat. → **S2**
- **Plynový kotel** — výměna za nový. → **S7** (v nabídce mistra CHYBÍ celé)
- **Schodiště** — reštaurace, odstranit skřípání; **zapečetit / ochránit PŘED** stavebními pracemi, ne na konci. → **S9**
- **Okna a dveře** — částečná reštaurace, výměna zamazek (zasklívacích lišt) + kompletní nátěr. → **S8**
- **Vývoz musoru + doprava + přenos materiálu.** → **S10** (VRN)

## Samo-přiznané mezery majitele (nad nabídku mistra)
- **Hodinové práce** — cca +50k (zapomenuto v nabídce).
- **Administrativa** — cca +50k.
- (mistr) **Hydroizolace** koupelny — „bude upřesněno", neoceněno (~15–25k).

## Pozn. k metodě
Jedna položka mistra (např. „Demontáž koupelny a WC", „Renovace koupelny") = **balík**
atomic operací, ne jeden katalogový kód. Viz Pattern 15 (Work-First, Catalog-Last) +
HK212 atomic decomposition. Atomic worklist je rozklad přesně tohoto scope.
