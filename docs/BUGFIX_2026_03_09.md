# BUGFIX 2026-03-09: Critical Production Issues

## 🔴 Проблемы

### 1. **Парсер не находит позиции (0 positions)**
**Симптомы:**
```
Total raw positions extracted: 0
format=UNKNOWN
```

**Причина:** Portal использует `workflowAStart()` который НЕ существует в `concreteAgentClient.js`

**Решение:** ✅ Заменён на прямой вызов Workflow C API (`/api/v1/workflow/c/upload`)

---

### 2. **Memory Overflow на Render**
**Симптомы:**
```
Web Service concrete-agent exceeded its memory limit
```

**Причина:** Парсинг больших PDF (58 страниц, 1.6MB) загружает всё в память

**Решение:** ✅ Добавлено ограничение на 50 страниц для PDF >20MB

---

### 3. **SQL ошибка: column position_instance_id does not exist**
**Симптомы:**
```
error: column pp.position_instance_id does not exist
code: '42703'
```

**Причина:** Неправильный SQL запрос в `position-instances.js:162`

**Решение:** ✅ Удалена несуществующая колонка из SELECT

---

### 4. **Multipart parsing error**
**Симптомы:**
```
Expected boundary character 45, got 91 at index 2
```

**Причина:** Неправильный Content-Type boundary при загрузке файлов

**Решение:** ✅ Используется правильный FormData с `form.getHeaders()`
