# SKRUŽ vs STOJKY vs PODPĚRNÁ KONSTRUKCE — kanonická terminologie

**Zdroje (autoritativní):**
- ČSN EN 12812:2008 (73 8108) — *Podpěrná lešení — Požadavky na provedení a obecný návrh*
- TKP 18 ŘSD (Betonové mosty a konstrukce) — kapitola 18.x *Bednění, skruže a lešení*
- DOKA CZ — Školení zákazníků (Ing. Martin Sosna, 09.01.2026): Staxo 100, Top 50
- PERI — VARIOKIT VST Product Catalogue, MULTIPROP technical docs
- ebeton.cz — T. Rotter: *Concrete bridges — history and future* (2008/4)
- Univerzita Pardubice — *Betonové mosty II* (skripta)
- beton.cz / asb-portal.cz — oborové slovníky

## 1) Oficiální hierarchie pojmů (ČSN + TKP 18)

```
DOČASNÉ STAVEBNÍ KONSTRUKCE (ČSN EN 12812, ČSN EN 12811, ČSN EN 1065)
│
├── PODPĚRNÁ LEŠENÍ / PODPĚRNÁ KONSTRUKCE  ← obecný pojem podle ČSN EN 12812
│   │  (= falsework / Traggerüst)
│   │  "dočasná konstrukce přenášející zatížení čerstvého betonu
│   │   a ostatních stavebních zatížení do únosného podkladu"
│   │
│   ├── STOJKY (ČSN EN 1065 — ocelové, ČSN EN 16031 — hliníkové)
│   │      = výsuvné stojky jako jednotlivé prvky
│   │      "individual adjustable props"
│   │
│   ├── PODPĚRNÉ VĚŽE / PODPĚRNÉ RÁMY
│   │      = rámové systémy z jednotlivých rámů a spojek
│   │      příklad: Staxo 100, Staxo 40, PERI UP Shoring Tower
│   │
│   └── SKRUŽE  ← speciální pojem z TKP 18 / mostního stavitelství
│          "podpěrná konstrukce nesoucí bednění mostovky
│           během betonáže, dokud beton nedosáhne potřebné pevnosti"
│          (typicky 50–100+ tun na sekci, statický návrh nutný)
│
├── PRACOVNÍ LEŠENÍ (ČSN EN 12811)
│   = pro přístup pracovníků, NE pro přenos svislých zatížení betonáže
│
└── BEDNĚNÍ (ČSN EN 13670, TKP 18)
    = forma vytvářející tvar betonu (desky + nosníky)
    NENÍ podpěrná konstrukce!
```

**Klíčový poznatek:**
- V obecné rovině (budovy, stropy) oficiálně používáme *podpěrné lešení* / *podpěrná konstrukce* / *stojky*.
- Slovo **„skruž" je specificky mostní pojem** — používá se v TKP 18 a v mostní praxi.
- DOKA pro Staxo 100 nepoužívá slovo "skruž" — používá **„nosná konstrukce"**.

## 2) SKRUŽ — co to přesně je (TKP 18, ebeton.cz, skripta UPCE)

**Skruž** = podpěrná konstrukce **specificky pro mostovky** (nosné konstrukce mostů). Rozdělení podle mostní praxe:

| Typ skruže | Význam | Použití |
|---|---|---|
| **Pevná skruž** | Stacionární věž/rám pod celým polem | Krátká pole, dostupný terén pod mostem |
| **Posuvná skruž** (MSS — spodní) | Pohybuje se po zemi/nosnících | Dlouhé mosty, více polí o stejném rozpětí |
| **Výsuvná skruž** (MSS — horní) | Zavěšené bednění na tyčích | Kde není prostor pod mostem |
| **Skruž UniKit (DOKA)** / **VARIOKIT VST (PERI)** | Modulární těžké nosné věže | Mosty, tunely, průmysl, demolice |

Citáty z autoritativních zdrojů:

- **TKP 18 (ŘSD)**: nadpis 18.x — *„Bednění, skruže a lešení"* — tři oddělené kategorie.
- **ebeton.cz (T. Rotter, 2008)**: *„Posuvné skruže byly vyvinuty v Německu v 60. letech. U nás byla posuvná skruž poprvé použita na mostě u Hvězdonic (na D1)…"*
- **asb-portal.cz (Doka)**: *„Realizace mostů na pevné skruži — nejběžnější podepření mostů pevnou skruží se využívá zejména na kratší mostní úseky."*
- **beton.cz — slovník pojmů**: *„Pevné skruže — modulární nosné konstrukce umožňují bednění různých profilů tunelů… typickým příkladem je systém Staxo 100 s únosností 1000 kN na nohu."*

**Proto:** když DOKA v brožuře Staxo 100 píše „nosná konstrukce" a asb-portal píše „pevná skruž Staxo 100" — **je to to samé**. Rozdíl je v tom, **pro co se to používá** a **kdo to pojmenovává**.

## 3) Co DOKA a PERI skutečně říkají ve svých katalozích

|---|---|---|---|
| **Staxo 100** | *Nosná konstrukce* (ne „skruž"!) | **100 kN (10 t) na nohu**, celkem 1000 kN/věž | Infrastruktura, energetika, mostovky, **těžké podpírání** |
| **Staxo 40** | *Rám Staxo výšky 120* / stropní podepření | do ~40 kN/noha | Stropní systémy s **Dokaflex 20** — bytová výstavba |

**DŮLEŽITÉ:** DOKA sama **nepoužívá slovo „skruž"** pro Staxo 100 ve svých školicích materiálech. Používá **„nosná konstrukce"**. ## 4) Jak správně mluvit s obchodním zástupcem DOKA/PERI

**Špatně (vypadá amatérsky):**
> „Potřebuju skruže na most."

Proč špatně: slovo pokrývá příliš mnoho věcí, DOKA se bude muset doptávat na všechno.

**Správně (3 věty které DOKA/PERI okamžitě zařadí):**

> „Mám mostovku výšky podepření **~6,5 m**, zatížení asi **80 kN na nohu**, potřebuju **systémové podpěrné věže (skruž)** pod betonáž mostovky. Chtěl bych cenový návrh na **Staxo 100 + Top 50** včetně **statického návrhu od výrobce**, dopravy, montáže, **pronájmu** na dobu X měsíců a demontáže."

Nebo pro PERI:

> „Potřebuju **heavy-duty shoring** pro mostovku. Výška ~30 m, zatížení na nohu ~600 kN. Uvažuji o **VARIOKIT VST** s **VRB** jako horizontální nosníky. Prosím o **návrh + statiku + cenu pronájmu**."

**Pro demolici (tvůj případ):**

> „Demolujeme mostní konstrukci, dílce **75–98 t**. Potřebuju **podpěrné skruže — zřízení a odstranění** pod mostovkou během řezání a snímání dílců jeřábem. Objem kolem **1200 m³ skruží**. Chci těžký systém (**UniKit / VARIOKIT VST**), **statický výpočet pro demoliční zatížení**, dopravu, montáž, **měsíční pronájem**, demontáž. **Kolik a za jakých podmínek?**"

## 5) Klasifikace pro kalkulátor STAVAGENT (implementace)

```typescript
/**
 * Canonical taxonomy aligned with ČSN EN 12812 + TKP 18 + DOKA/PERI catalogs.
 * Source: SKRUZ_TERMINOLOGIE_KANONICKA.md
 */

// Obecný pojem podle ČSN EN 12812
type PodpernaKonstrukce = {
  kategorie: 'skruz' | 'stojky' | 'podperne_leseni';
  system: FormworkSystem;
  nosnost_kN_per_leg: number;
  vyska_m: number;
  staticky_navrh_od_vyrobce: boolean;
};

// Specificky mostní (TKP 18) — vždy těžký systém, statický návrh nutný
type Skruz =
  | { typ: 'pevna_skruz'; system: 'staxo_100' | 'unikit_doka' | 'variokit_vst' | 'peri_up_shoring' }
  | { typ: 'posuvna_skruz_spodni' /* MSS dolní */ }
  | { typ: 'posuvna_skruz_horni' /* MSS horní, Berd */ }
  | { typ: 'vysuvna_skruz' /* letmá betonáž, vozík */ };

// Pro budovy, lehké zatížení
type Stojky =
  | { system: 'staxo_40'; dokaflex: 'dokaflex_20' | 'dokaflex_30tablex' }
  | { system: 'multiprop'; mrk_ram: boolean }
  | { system: 'stojka_cskos_en_1065' /* generické výsuvné */ };

// Rozhodovací logika — KDY skruž, KDY stojky
function klasifikujPodperu(el: ElementProfile): 'skruz' | 'stojky' {
  // Mostní prvky → vždy skruž (TKP 18 požaduje statický návrh)
  const MOSTNI_PRVKY = [
    'mostovka_deska', 'mostovka_nosnik',
    'driky_piliru', 'opery_ulozne_prahy',
    'kridla_mostu', 'prechodova_deska',
  ];
  if (MOSTNI_PRVKY.includes(el.element_type)) return 'skruz';

  // Výška podepření >5 m → skruž (nad tímto limit Staxo 40 / MULTIPROP
  // už vyžadují věžovou sestavu ~ prakticky skruž)
  if (el.podperna_vyska_m > 5) return 'skruz';

  // Zatížení na nohu >50 kN → skruž (nad limit jednotlivých stojek)
  if (el.zatizeni_kN_per_leg > 50) return 'skruz';

  // Demolice → vždy skruž (těžké dynamické zatížení)
  if (el.kontext === 'demolice') return 'skruz';

  // Jinak stojky (bytová výstavba, standardní stropy)
  return 'stojky';
}

// UI labeling — ukázat uživateli správně
const UI_LABEL = {
  skruz: {
    cs: 'Skruž (systémová podpěrná věž)',
    doka: 'Staxo 100 / UniKit',
    peri: 'VARIOKIT VST',
    nosnost_info: 'až 100 kN/noha (Staxo 100) / až 600 kN/noha (VST)',
    staticky_vypocet: 'Nutný návrh od výrobce',
    pronajem_kc_m2_mesic: [280, 350], // orientační
  },
  stojky: {
    cs: 'Stojky (stropní podpěry)',
    doka: 'Staxo 40 + Dokaflex 20',
    peri: 'MULTIPROP + MRK',
    nosnost_info: 'do ~40 kN/noha (Staxo 40) / do 100 kN/prop (MP)',
    staticky_vypocet: 'Standardní statický návrh zhotovitele',
    pronajem_kc_m2_mesic: [120, 180], // orientační
  },
};
```

## 6) Co DOKA/PERI očekávají ve výzvě k cenové nabídce

Pro skruž (mostovka) potřebují znát:

1. **Typ konstrukce** (most nová / demolice / tunel / průmysl)
2. **Podpěrná výška** (od rostlého terénu k dolní ploše bednění)
3. **Zatížení** — buď v kN/m² (plošné od betonu + zatížení) nebo v kN/noha (pokud už víš rozteč)
4. **Rozměry** — délka × šířka podepření (m²)
5. **Termíny** — montáž od/do, pronájem měsíců, demontáž
6. **Přístup** — zda je ruční montáž / jeřábem / helikoptérou
7. **Statický výpočet** — vyžadujete návrh od výrobce? (pro mosty a demolice ANO)
8. **Místo** — ovlivňuje dopravu (v ČR DOKA Praha/Brno, PERI Jesenice u Prahy)

Pro stojky (strop v budově):

1. **Plocha podepření** (m²)
2. **Tloušťka stropu** (beton → zatížení) + provozní zatížení
3. **Světlá výška**
4. **Termín — pronájem do kdy**
5. **Jaké bednění stropní?** (ovlivňuje rastr stojek)

## 7) TL;DR pro rychlou orientaci

| Otázka | Odpověď |
|---|---|
| Je „skruž" nadřazený pojem? | **Ne.** Nadřazený je *podpěrné lešení / podpěrná konstrukce* (ČSN EN 12812). |
| Je každá skruž podpěrná konstrukce? | **Ano.** |
| Je každá podpěrná konstrukce skruž? | **Ne.** Stojky pod stropem nejsou skruž. |
| Kde se termín „skruž" oficiálně používá? | **TKP 18 ŘSD** (mosty), mostní literatura, projekty mostů, rozpočty mostů. |
| Používá DOKA slovo „skruž" pro Staxo 100? | **V technických dokumentech Staxo 100 — ne**, používá *„nosná konstrukce"*. V mostní aplikaci ano. |
| Je Staxo 40 skruž? | **Ne**, Staxo 40 jsou stojky pro stropy budov (Dokaflex 20). |
| Je Staxo 100 skruž? | **Ano, když se nasadí pod mostovku.** Jinak „nosná konstrukce 100 kN/noha". |
| Co je PERI ekvivalent Staxo 100? | **VARIOKIT VST** (Heavy-Duty Shoring Tower). |
| Co je PERI ekvivalent Staxo 40? | **MULTIPROP + MRK rám** (se shoring tower). |
| Top 50 je skruž? | **Ne.** Top 50 je *nosníkové bednění* (stěn). Je to bednění, ne podpora. |
| Co dělat pro demolici mostu? | Vždy **skruž** (heavy-duty), **statický návrh od výrobce**, samostatné řádky: doprava / montáž / pronájem / demontáž. |

## 8) Reference pro přípravu rozpočtu (OTSKP / ÚRS)

V OTSKP a ÚRS se skruže / stojky / bednění rozpočtují jako **samostatné položky** s kódy. Obvyklé kategorie:

- **Bednění** — kód OTSKP: 5. číslice = 5 (bednění). Např. `311356xxx` = bednění svislých konstrukcí.
- **Podpěrné lešení / skruž** — samostatné položky, často uvedeno jako *„skruž — zřízení a odstranění"* s m² / m³ / t.
- **Pronájem** — odděluje se od montáže a demontáže. Typicky samostatná položka *„Pronájem podpěrné skruže/lešení"* s jednotkou **m²·měsíc** nebo **m³·měsíc**.
- **Statický výpočet / projekt od výrobce** — samostatná položka *„Projekční práce — návrh skruže"*.

**Pro tvůj kalkulátor:** pronájem skruže/stojek **vždy rozděl do 3 řádků**:
1. Zřízení (montáž + doprava na stavbu)
2. Pronájem (měsíční sazba × počet měsíců)
3. Odstranění (demontáž + doprava ze stavby)

Plus volitelně 4. řádek: **statický návrh** (jednorázová cena od výrobce, typicky 15–50 tis. Kč pro běžnou mostovku, 100+ tis. Kč pro demolici).
