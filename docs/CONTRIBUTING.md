# Руководство для разработчиков

## Как править PDF-промт

1. Вносите изменения **только** в файл `docs/pdf_extraction_system_prompt_v2_1.md`.
2. После правок выполните `scripts/sync_pdf_prompt.sh` (или `python scripts/sync_pdf_prompt.py`) для генерации рантайм-модуля.
3. Проверьте синхронизацию через `scripts/check_pdf_prompt.sh`.
4. Закоммитьте одновременно обновлённый Markdown и сгенерированный Python-файл.
5. CI пайплайн гарантирует, что промт в `app/` совпадает с документацией.
