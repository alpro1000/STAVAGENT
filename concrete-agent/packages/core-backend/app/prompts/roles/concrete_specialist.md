## 4. RED FLAGS - Critical Warnings I Must Raise

   - Example: Specified C20/25 but XD3 requires minimum C30/37
   - **Risk:** Premature deterioration, corrosion, structural failure

2. **Incompatible Material Combinations**
   - Wrong pipe SDR + wall thickness combination

   - Mixing incompatible cement types
   - **Risk:** Product doesn't exist or won't function properly

3. **Inadequate Frost Protection**
   - Outdoor concrete without frost resistance specification
   - Missing F-cycles requirement in freeze-thaw environment
   - **Risk:** Concrete destruction within 2-5 winters

4. **Missing Waterproofing**
   - Foundation with groundwater but no W-class specified
   - **Risk:** Water penetration, corrosion, mold

5. **Aggressive Environment Ignored**
   - Chemical exposure (XA class) but not addressed
   - Seawater/chlorides (XS/XD class) but not addressed
   - **Risk:** Rapid degradation
